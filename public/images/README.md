# Images Mockup - ShopFlow

Ce dossier contient les images utilisées pour la page d'accueil de ShopFlow.

## Images requises

### 1. mockup-desktop.png
- **Dimensions recommandées** : 1920x1200 pixels (ratio 16:10)
- **Format** : PNG ou JPG
- **Description** : Capture d'écran du tableau de bord ShopFlow en version desktop
- **Conseil** : Prenez une capture de la page dashboard avec des données d'exemple

### 2. mockup-mobile.png  
- **Dimensions recommandées** : 390x844 pixels (ratio 9:19, iPhone 14)
- **Format** : PNG ou JPG
- **Description** : Capture d'écran de l'application en version mobile
- **Conseil** : Utilisez les DevTools Chrome (F12) pour simuler un iPhone et prendre la capture

## Comment ajouter les images

1. Prenez des captures d'écran de votre application
2. Redimensionnez-les aux dimensions recommandées
3. Placez-les dans ce dossier avec les noms exacts :
   - `mockup-desktop.png`
   - `mockup-mobile.png`

## Placeholder

Si les images ne sont pas présentes, un placeholder avec le logo "SF" sera affiché automatiquement.

## Outils recommandés

- **Captures d'écran** : Snipping Tool (Windows), Screenshot (Mac)
- **Redimensionnement** : Photoshop, GIMP, ou [Squoosh](https://squoosh.app/)
- **Simulation mobile** : Chrome DevTools (F12 > Toggle device toolbar)
